# Noddy2-2023
Node.js Express, MongoDB with Winston and Morgan for 2023 class update

## Setup
1 - Clone the repo

2 - Add dependencies

   npm install

3 - Copy env.DEV config to .env.DEV and make edits for your MongoDB connection and server port

   cp env.DEV .env.DEV

4 - Run Application

   node server.js
